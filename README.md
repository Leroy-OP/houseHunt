# House Hunt

## Overview

House Hunt is a project aimed at helping users find their ideal homes by providing lists of available properties based on user preferences. This application leverages advanced filtering and searching capabilities for a seamless user experience.

## Features
- **Property Search**: Search homes based on location, price, and other criteria.
- **User Profiles**: Save your favorite properties and find homes that match your preferences.
- **Notifications**: Get alerted when new properties that fit your criteria are listed.

## Installation

To install this application, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Leroy-OP/houseHunt.git
   ```
2. Navigate to the project directory:
   ```bash
   cd houseHunt
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

After installation, start the application with:
```bash
npm start
```
You can then access the application at `http://localhost:3000`.
